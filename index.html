NASA Astronomy Picture of the Day (APOD) API

Project Overview

This project demonstrates how to use NASA's APOD API to fetch daily astronomy pictures. It includes clear instructions for Windows 11 users using PowerShell, Python, and JavaScript. This portfolio piece showcases my technical writing skills, including clear step-by-step instructions, sample code, and visual outputs.

Setup Instructions

1. Get a free NASA API Key: https://api.nasa.gov/

2. Prerequisites:

  • Windows 11
  • PowerShell (preinstalled)
  • Python 3 (optional)
  • Web browser for HTML/JS example

PowerShell Example

Fetch today’s APOD and display JSON:

$response = Invoke-RestMethod -Uri "https://api.nasa.gov/planetary/apod?api_key=rxCidX2jSbVzqX4Eg7mgVu9SAFf2RSXq0NzQXHLR"
$response | ConvertTo-Json -Depth 5

Download the image to your desktop:

Invoke-WebRequest -Uri $response.url -OutFile "$env:USERPROFILE\Desktop\NASA_APOD.jpg"

Python Example

import requests

api_key = "rxCidX2jSbVzqX4Eg7mgVu9SAFf2RSXq0NzQXHLR"
url = f"https://api.nasa.gov/planetary/apod?api_key={api_key}"

response = requests.get(url).json()

print("Title:", response["title"])
print("Date:", response["date"])
print("Explanation:", response["explanation"])
print("Image URL:", response["url"])

JavaScript / Web Example

Create a simple webpage to display APOD:

<!DOCTYPE html>
<html>
<head></head>
<body>
<script>
const apiKey = "rxCidX2jSbVzqX4Eg7mgVu9SAFf2RSXq0NzQXHLR";
fetch(`https://api.nasa.gov/planetary/apod?api_key=${apiKey}`)
  .then(res => res.json())
  .then(data => {
    document.body.innerHTML = `
      <h2>${data.title} (${data.date})</h2>
      <img src="${data.url}" alt="NASA APOD" width="600">
      <p>${data.explanation}</p>
    `;
  });
</script>
</body>
</html>
