NASA Astronomy Picture of the Day – Simulated Demo

Project Overview

This project demonstrates how a NASA APOD viewer works by simulating random astronomy pictures using local images and JSON data. This portfolio piece showcases technical writing, clear instructions, sample code, and visual outputs.

Setup Instructions

  • Windows 11
  • Web browser for HTML/JS example
  • Local folder containing images (/images) and JSON data (/data/apod-data.json)

Usage Examples

PowerShell
Python
JavaScript

PowerShell Example

# Simulated example using local JSON
$response = Get-Content "data/apod-data.json" | ConvertFrom-Json
$response | ConvertTo-Json -Depth 5

Python Example

import json

with open("data/apod-data.json") as f:
    data = json.load(f)

print(data[0]["title"])
print(data[0]["date"])
print(data[0]["explanation"])
print(data[0]["url"])

JavaScript Example

// Simulated APOD fetch using local JSON
fetch("data/apod-data.json")
  .then(res => res.json())
  .then(data => console.log(data));
